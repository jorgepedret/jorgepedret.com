It bugged my mind for a bit so I decided to figure out how they did it. It turns out it’s a lot more simple than you think, all you need is two images and CSS.

See the demo Download the demo files

Here’s how you can easily create the effect:

Start by creating the transition image

  1. Create a 1200 x 300 pixels image in Photoshop with transparent background.
  2. Create the images that you want to see at the top of the page and make sure they’re only in #666.
  3. Then, create the images that you want to see at the bottom of the page and make sure they’re only in #000.
  4. Save it as a PNG named bottom-bg.png, making sure there’s no background color. The only two colors that can exist in this image are #666 and #000.
The two top and bottom images can’t be touching each other, they have to be separate, otherwise the effect will not work. They can be very close though.
This is the design that Think Geek used: https://www.thinkgeek.com/images/sitewide/backgrounds/robozombies7.png

Next, create the background image

  1. Create a 10 x 700 pixels image in Photoshop
  2. Select the gradient tool and configure the gradient to be from #000 to #666
  3. Vertically drag it from the very top to the middle of the image approximately.
  4. Save that as a PNG file named body-bg.png and you’re done with the background image.
Once you understand the way this effect works, you will be able to use whichever colors you want. But for now we’re sticking to #000 and #666

And finally, write the CSS and HTML

Let’s start with the HTML:

<!DOCTYPE html>
<html lang="en">
<head>
</head>
<body>
<div class="fancywrap">
	<div id="wrap">
		<article>
		<!-- Your content here -->
		</article>
	</div>
</div>
</body>
</html>
This is a simple structure for this demo, but you can adapt it to your own project.

For the styles, you’re going to attach to your body tag the background image that you created in step 2 and make the color background #000.

body{
	background: #000 url(body-bg.png) left bottom repeat-x;
}

Next step is styling the .facywrap class by attaching the transition image to it. And the secret for this effect to work is to use the CSS property background-attachment: fixed; like this:

.fancywrap{
	background: url('bottom-bg.png') repeat-x;
	background-position: center bottom;
	background-attachment: fixed;
}

See the full demo code in Github: https://gist.github.com/937679

That’s it! Once you understand how it works, you will be able to choose your own color combination other than the boring #000 and #666.

Javascript is the most widely deployed language ion the world. People are creating more apps in JavaScript than before because:

  • Computers are faster
  • All the raising JavaScript libraries that standardize and make JS a lot cooler

JavaScript and PHP similarities

  • Dynamic and loose
  • C-like syntax
  • Curly braces
  • if, else, for, etc.
  • Garbage collection
  • Post-web

JavaScript and PHP differences

  • Everything is an object
  • Objects and inheritance
  • Scope and this

Lint

  • Bad parts are simple mistakes
  • JSLint, JavaScript Lint, JSHint
  • Auto lint

WordPress

The function wp_enque script helps you load common libraries like jQuery, Scriptaculous, jQUI, TinyMCE, and it lets let load your own js files. You can also specify if you want your files to load in the footer to optimize loading speed . Thanks @apike for answering my question.

JavaScript Libaries

  • DOM (jQuery, Prototype) — Most popular libraries for manipulating the DOM
  • MVC (Backbone, JSMBC, Spine) — Libraries that helps you write JavaScript in a more organized and MVC way
  • Everything (ExtJS, YUI, SproutCore)  — They do everything
  • microjs.com — Recently launched library where you can chose what you want

Debugging JavaScript

  • console.log(), and console.trace()
  • stack traces, step debugging

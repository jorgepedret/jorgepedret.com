Jorge Pedret » reminder http://jorgepedret.com Freelance Web Developer + Front End Designer Fri, 24 Aug 2012 16:34:00 +0000 en-US hourly 1 http://wordpress.org/?v=3.4.1 Use jQuery to select child element http://jorgepedret.com/web-development/use-jquery-to-select-child-element/ http://jorgepedret.com/web-development/use-jquery-to-select-child-element/#comments Fri, 19 Feb 2010 01:12:30 +0000 Jorge Pedret http://jorgepedret.com/blog/?p=45 Suppose you have the following structure:

<div class="section">
	<h3 class="title">Title</h3>
	<span class="description">...</span>
</div>
<div class="section">
	<h3 class="title">Title</h3>
	<span class="description">...</span>
</div>

This code will let you access the child elements of ‘.section’ when you use an ‘each’ function

<script type="text/javascript">
$(function(){
	$('.section').each(function(i,element){
		$('description',this).fadeOut(100).fadeIn(100).fadeOut(100).fadeIn(100);
	});
});
</script>

I hope this helps someone, I kept forgetting how to do this, so this is mostly a reminder for myself.

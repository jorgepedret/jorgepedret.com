  1. Clone all the remote WordPress files in your local computer.
  2. Clone the remote WordPress database into your local computer.
  3. Update the wp-config.php file with your local MySQL access info and database name.
  4. Replace the old domain with the new domain in your local MySQL by running this commands:
UPDATE wp_options SET option_value = REPLACE(option_value, 'http://old.domain.com', 'http://new.domain.com');
UPDATE wp_posts SET post_content = REPLACE(post_content, 'http://old.domain.com', 'http://new.domain.com');

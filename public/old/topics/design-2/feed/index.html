He show us how to make Photoshop Layer Styles separate into their own layer and some reasons why we want to do that.

This is a technique I’ve never seen before, so if you don’t know what I’m talking about, just check out the video.

Here’s a quick step-by-step list:

  1. In Photoshop, select a layer with styles assigned to it.
  2. On the main menu, go to Layer > Layer Style > Create Layer (Click OK on the message prompt).
  3. Now all the styles are layers!
